## 📓 关键知识笔记：

- GitHub 从 2021 起废弃密码推送，必须使用 **Token** 或 **SSH**
- Token 权限建议仅勾选 `Repository → Contents: Read and write`
- 项目结构采用阶段划分法（Stage1~5）+ `daily/` 实时记录，利于持久跟踪成长
- Markdown (`.md`) 文件可使用 VS Code 实时编辑与预览（推荐插件：Markdown All in One）
